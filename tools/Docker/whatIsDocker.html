Docker常用命令

 docker run -it --rm alpine:latest /bin/sh
  • 参数解析：
    • -it： -i参数，是用于提供交互操作；-t是提供一个终端。总体就是提供一个交互式的终端，用于输入命令并查看返回结果。
    • --rm： 在容器退出后直接删除该容器。
    • /bin/sh： shell模式。可以简写sh，或其他shell类型：/bin/bash、bash等。
