docker镜像导入及导出

可用于将docker镜像下载到本地，并在未连接互联网的节点上进行恢复。

下载镜像

在一台可以联网并执行docker pull的设备上，将所需的镜像下载至本地。

xx为所需要拉取的镜像名称，如mariadb，busybox等。

v1.0代表所要拉取的版本号，如mariadb的10.11.5等。不填写版本号，则默认为latest。

docker pull xx:v1.0
